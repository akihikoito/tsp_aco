==================================================
    蟻コロニー最適化による店舗巡回最適経路探索
                                    by AKIHIKO ITO
==================================================

*** プログラム概要 *******************************

指定地名および検索キーワードから
指定地名近傍のレストラン店舗を検索。

指定地から出発し対象店舗を巡回する
最短経路を蟻コロニー最適化により計算し
得られた最適経路を地図上にプロットする。

レストラン店舗検索にぐるなびAPIを利用。

地名検索や店舗間距離の取得、地図作成に
Google Map Platformを利用する。

また地図表示簡易化のためfoliumを利用する。

*** 留意点 ***************************************

ぐるなびAPI および Google Map Platform の
アクセスキーを取得し、webapi.py コード内の
keyidを自身のキーに書き換えて使用すること。

Google Map Platform Distance Matrix API の
無料アクセス回数上限は現在40,000となっている。

検索結果の店舗数が100を超えると
数回プログラムを走らせただけで上限に達するため
店舗数が50店舗以下での計算をおすすめする。
